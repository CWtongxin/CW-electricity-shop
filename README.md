# 云商店 - 完整的电商网站系统

## 项目简介

这是一个使用原生 HTML5、CSS3 和 JavaScript 开发的现代化云商店电商平台。项目实现了完整的商品展示、购物车、订单结算等电商核心功能，采用响应式设计，支持 PC、平板和移动端访问。

## 技术栈

- **HTML5** - 语义化标签构建页面结构
- **CSS3** - 现代化样式设计，包含动画效果和响应式布局
- **JavaScript (ES6+)** - 原生 JS 实现所有交互逻辑，无需框架

## 项目结构

```
商店/
├── index.html          # 首页（轮播图、热门商品、新品推荐）
├── list.html           # 商品列表页（筛选、排序、分页）
├── detail.html         # 商品详情页（规格选择、评价展示）
├── cart.html           # 购物车页面（数量修改、批量操作）
├── checkout.html       # 结算页面（地址管理、支付方式）
├── style.css           # 全局样式文件
├── script.js           # 核心 JavaScript 逻辑
└── README.md           # 项目说明文档
```

## 数据说明

本项目使用模拟数据进行演示：
- 商品数据存储在 `script.js` 中的 `products` 数组
- 购物车数据使用 `localStorage` 持久化存储
- 用户数据使用 `localStorage` 模拟登录状态
- 地址数据使用内存数组模拟

## 主要函数说明

### 导航栏相关
- `handleSearch()` - 处理搜索功能
- `showLoginModal()` - 显示登录弹窗
- `updateCartCount()` - 更新购物车角标

### 轮播图相关
- `initCarousel()` - 初始化轮播图
- `changeSlide(direction)` - 切换幻灯片
- `goToSlide(index)` - 跳转到指定幻灯片

### 商品相关
- `loadProducts(category, sort, minPrice, maxPrice)` - 加载商品列表
- `renderProducts(productList, containerId)` - 渲染商品列表
- `createProductCard(product)` - 创建商品卡片
- `loadProductDetail(productId)` - 加载商品详情

### 购物车相关
- `addToCart(productId, quantity, specs)` - 添加到购物车
- `loadCart()` - 加载购物车页面
- `updateCartItemQuantity(index, delta)` - 更新商品数量
- `deleteCartItem(index)` - 删除商品
- `checkout()` - 去结算

### 结算相关
- `loadCheckout()` - 加载结算页面
- `selectAddress(index, element)` - 选择地址
- `submitOrder()` - 提交订单
- `saveAddress(event)` - 保存新地址

### 工具函数
- `showToast(message, duration)` - 显示提示弹窗
- `confirm(message)` - 确认对话框
- `formatDate(date)` - 格式化日期
