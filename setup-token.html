🔑 GitHub Token 配置

3 分钟完成配置，开启云存储功能

步骤 1：访问 Token 创建页面

点击下方按钮，在新窗口打开 GitHub Token 设置页面

步骤 2：创建新 Token

1. 点击 "Generate new token"
2. 选择 "Generate new token (classic)"
3. Note 填写：My Shop
4. Expiration 选择：No expiration

步骤 3：勾选权限（重要！）

⚠️ 必须勾选这两个权限才能正常使用！

步骤 4：复制 Token

点击 "Generate token" 后，会生成类似这样的 Token：
ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

⚠️ 立即复制！页面刷新后就看不到了！

✅ 配置成功！即将跳转到商店首页...
